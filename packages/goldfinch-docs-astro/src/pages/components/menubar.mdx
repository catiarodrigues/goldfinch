A horizontal navigation bar of icon buttons with tooltips — familiar for desktop-style toolbar interfaces.

## Installation

### Barrel

```tsx
import { MenuBar } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { MenuBar } from "@catiarodrigues/goldfinch/components/menubar";
```

## Usage

```tsx
import { MenuBar } from "@catiarodrigues/goldfinch";

export default function Example() {
  const items = [
    { icon: <BoldIcon />, tooltip: "Bold", onClick: () => {} },
    { icon: <ItalicIcon />, tooltip: "Italic", onClick: () => {} },
  ];
  return <MenuBar options={items} isActive={1} />;
}
```

## Examples

<DemoFrame title="Menubar">

Provide an `options` array where each item has an `icon`, `tooltip`, and `onClick` handler. Use `isActive` to highlight the current selection.

<Callout type="info">
  Menubar supports keyboard navigation with arrow keys between options.
</Callout>
