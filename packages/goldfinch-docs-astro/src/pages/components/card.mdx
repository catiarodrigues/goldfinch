Group related content in an elevated container that sits above the base surface — useful for cards, panels, and callout blocks.

## Installation

### Barrel

```tsx
import { Card } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { Card } from "@catiarodrigues/goldfinch/components/card";
```

## Usage

```tsx
import { Card } from "@catiarodrigues/goldfinch";

export default function Example() {
  return (
    <Card className="p-6">
      <h3 className="text-lg font-semibold">Card Title</h3>
      <p className="text-goldfinch-muted">Card content goes here.</p>
    </Card>
  );
}
```

Card applies a consistent elevation shadow and padding. Use it to group related content in a distinct visual container.

For layered layouts, use the compound sub-components:

```tsx
<Card>
  <Card.Secondary>Next Steps</Card.Secondary>
  <Card.Primary>Get started with Goldfinch</Card.Primary>
</Card>
```

<Callout type="info">
  Card uses a medium elevation level by default. Nest within containers for layered layouts.
</Callout>
