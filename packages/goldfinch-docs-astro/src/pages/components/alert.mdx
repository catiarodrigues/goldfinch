Display prominent notices — release notes, system warnings, validation summaries — with optional icon, title, description, and action slot.

## Installation

### Barrel

```tsx
import { Alert } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { Alert } from "@catiarodrigues/goldfinch/components/alert";
```

## Usage

```tsx
import { Alert } from "@catiarodrigues/goldfinch";

export default function Example() {
  return (
    <Alert title="Update available" description="A new version is ready to install." />
  );
}
```

Choose a variant that matches the urgency — `default` for general info, `alert` for time-sensitive callouts, `error` for failures, `secondary` for neutral messages.

```tsx
<Alert variant="default" title="Info" description="General information." />
<Alert variant="alert" title="Warning" description="Time-sensitive notice." />
<Alert variant="error" title="Error" description="Critical issue." />
<Alert variant="secondary" title="Note" description="Neutral message." />
```

<Callout type="warning">
  Alerts are persistent and should only be used for information that requires immediate attention.
</Callout>
