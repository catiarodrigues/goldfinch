Choose from a predefined list — pass an array of `{ label, value }` objects via `items`, or compose custom option markup with `Select.Option` children.

## Installation

### Barrel

```tsx
import { Select } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { Select } from "@catiarodrigues/goldfinch/components/select";
```

## Usage

```tsx
import { Select } from "@catiarodrigues/goldfinch";

export default function Example() {
  const [value, setValue] = useState("option-1");
  const options = [
    { label: "Option 1", value: "option-1" },
    { label: "Option 2", value: "option-2" },
  ];
  return (
    <Select
      items={options}
      value={value}
      onValueChange={setValue}
      placeholder="Select an option"
    />
  );
}
```

## Examples

<LiveSelect title="Select" />

Provide options via the `items` prop. Use `value` and `onValueChange` for controlled mode, or `defaultValue` for uncontrolled.

```tsx
<Select
  items={[
    { label: "Option 1", value: "1" },
    { label: "Option 2", value: "2" },
  ]}
  placeholder="Choose..."
/>
```

<LiveSelectItems title="Items" />

For custom option markup, use the compound children API with `Select.Option`, `Select.Group`, `.GroupLabel`, and `.Separator`.

```tsx
<Select placeholder="Pick one">
  <Select.Group>
    <Select.GroupLabel>Fruits</Select.GroupLabel>
    <Select.Option value="apple">Apple</Select.Option>
    <Select.Option value="banana">Banana</Select.Option>
  </Select.Group>
</Select>
```

<Callout type="info">
  Select items can be defined as an object map or an array of `{ label, value }` objects.
</Callout>
