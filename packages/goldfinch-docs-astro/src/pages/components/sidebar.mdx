Full-featured application sidebar with collapsible sections, nested menu items, active state tracking, keyboard navigation, and resizable panels.

## Installation

### Barrel

```tsx
import { Sidebar } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { Sidebar } from "@catiarodrigues/goldfinch/components/sidebar";
```

## Usage

```tsx
import { Sidebar } from "@catiarodrigues/goldfinch";

export default function Example() {
  const items = [
    { id: "dashboard", label: "Dashboard", icon: <DashboardIcon /> },
    {
      id: "settings",
      label: "Settings",
      icon: <SettingsIcon />,
      items: [
        { id: "profile", label: "Profile" },
        { id: "account", label: "Account" },
      ],
    },
  ];
  return <Sidebar items={items} activeItemId="dashboard" onItemSelect={(id) => console.log(id)} />;
}
```

## Examples

Define navigation items via the `items` prop. Track the active item with `activeItemId` and handle selection with `onItemSelect`.

```tsx
<Sidebar
  items={[
    { id: "home", label: "Home", icon: <HomeIcon /> },
    { id: "settings", label: "Settings", icon: <GearIcon /> },
  ]}
  activeItemId="home"
  onItemSelect={(id) => navigate(id)}
/>
```

<Callout type="info">
  Sidebar items can include icons, labels, and nested sub-items for hierarchical navigation.
</Callout>
