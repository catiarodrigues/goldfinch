A form label that can show an "(Optional)" indicator and an info tooltip.

## Installation

### Barrel

```tsx
import { Label } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { Label } from "@catiarodrigues/goldfinch/components/label";
```

## Examples

<DemoFrame title="Label">
  <div className="flex flex-col gap-3">
    <Label>Email</Label>
    <Label showOptional>Middle Name</Label>
    <Label tooltip="We'll use this to verify your account.">Phone</Label>
  </div>
</DemoFrame>

