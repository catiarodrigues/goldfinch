A modal overlay that captures focus and blocks page interaction until dismissed — use for confirmations, quick forms, or detail views.

## Installation

### Barrel

```tsx
import { Modal } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { Modal } from "@catiarodrigues/goldfinch/components/modal";
```

## Usage

```tsx
import { Modal, Button } from "@catiarodrigues/goldfinch";
import { useState } from "react";

export default function Example() {
  const [open, setOpen] = useState(false);
  return (
    <Modal.Root open={open} onOpenChange={setOpen}>
      <Modal.Trigger>
        <Button>Open Modal</Button>
      </Modal.Trigger>
      <Modal>
        <Modal.Title>Hello</Modal.Title>
        <Modal.Description>Modal content here.</Modal.Description>
        <Modal.Close>
          <Button>Close</Button>
        </Modal.Close>
      </Modal>
    </Modal.Root>
  );
}
```

Control the modal with `open` and `onOpenChange`. Provide a `title` for accessibility. Four sizes available: `sm`, `base`, `lg`, `xl`.

```tsx
<Modal size="sm">...</Modal>
<Modal size="base">...</Modal>
<Modal size="lg">...</Modal>
<Modal size="xl">...</Modal>
```

<Callout type="warning">
  Always provide a `title` for screen readers. The modal traps focus while open to meet accessibility standards.
</Callout>
