Collect API keys, tokens, and other secrets with the value masked by default and a toggle to reveal it on demand.

## Installation

### Barrel

```tsx
import { SensitiveInput } from "@catiarodrigues/goldfinch";
```

Or as a compound sub-component:

```tsx
import { Input } from "@catiarodrigues/goldfinch";
// Input.Sensitive is available as a sub-component
```

### Granular

```tsx
import { SensitiveInput } from "@catiarodrigues/goldfinch/components/sensitive-input";
```

## Usage

```tsx
import { SensitiveInput, Field } from "@catiarodrigues/goldfinch";

export default function Example() {
  return (
    <Field label="API Key">
      <SensitiveInput placeholder="sk-..." aria-label="API Key" />
    </Field>
  );
}
```

## Examples

<LiveSensitiveInput title="Sensitive Input" />

Use this component instead of a standard Input when the value should not be visible on screen by default.

```tsx
<SensitiveInput placeholder="sk-proj-..." aria-label="API Key" />
```

<Callout type="warning">
  SensitiveInput masks the display but does not encrypt or secure the value in memory. Treat the value with appropriate security measures.
</Callout>
