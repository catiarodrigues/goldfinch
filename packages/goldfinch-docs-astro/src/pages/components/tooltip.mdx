Annotate icons, buttons, and controls with concise labels that appear on hover (mouse) or focus (keyboard) and auto-position around the trigger.

## Installation

### Barrel

```tsx
import { Tooltip } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { Tooltip } from "@catiarodrigues/goldfinch/components/tooltip";
```

## Usage

```tsx
import { Tooltip, Button } from "@catiarodrigues/goldfinch";

export default function Example() {
  return (
    <Tooltip label="Delete item">
      <Button variant="ghost" aria-label="Delete">
        <TrashIcon />
      </Button>
    </Tooltip>
  );
}
```

## Examples

<LiveTooltip title="Tooltip" />

Wrap the trigger element as `children` and provide the tooltip text via the `label` prop. The tooltip positions itself automatically.

```tsx
<Tooltip label="Settings">
  <button aria-label="Settings"><GearIcon /></button>
</Tooltip>
```

<Callout type="info">
  Tooltips appear on hover for mouse users and on focus for keyboard users, ensuring accessibility across input methods.
</Callout>
