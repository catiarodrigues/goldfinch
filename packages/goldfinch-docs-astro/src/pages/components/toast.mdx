Display brief, auto-dismissing notifications anchored to the bottom of the viewport. Supports programmatic show/dismiss via an imperative manager.

## Installation

### Barrel

```tsx
import { Toast, createGoldfinchToastManager } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { Toast, createGoldfinchToastManager } from "@catiarodrigues/goldfinch/components/toast";
```

## Usage

```tsx
import { Toast, createGoldfinchToastManager, Button } from "@catiarodrigues/goldfinch";

const toastManager = createGoldfinchToastManager();

export default function Example() {
  return (
    <Toast toastManager={toastManager}>
      <Button onClick={() => toastManager.show("Hello!")}>Show Toast</Button>
    </Toast>
  );
}
```

## Examples

<LiveToast title="Toast" />

Wrap your application in Toast. Use the `toastManager` prop to use an external manager, or let it manage toasts internally.

```tsx
const manager = createGoldfinchToastManager();

<Toast toastManager={manager}>
  <App />
</Toast>

// Later, imperatively:
manager.show("Notification message");
```

<Callout type="warning">
  Only one Toast instance should be rendered in the application tree to avoid duplicate notifications.
</Callout>
