Indicate ongoing processing with an animated spin animation — available in `sm`, `base`, and `lg` sizes for different contexts.

## Installation

### Barrel

```tsx
import { Spinner } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { Spinner } from "@catiarodrigues/goldfinch/components/spinner";
```

## Usage

```tsx
import { Spinner, Button } from "@catiarodrigues/goldfinch";

export default function Example() {
  return (
    <Button disabled>
      <Spinner size="sm" aria-label="Loading" />
      Saving...
    </Button>
  );
}
```

Choose a size that fits the context — `sm` for inline use, `base` for buttons, and `lg` for full-page loading states.

```tsx
<Spinner size="sm" aria-label="Loading" />
<Spinner size="base" aria-label="Loading" />
<Spinner size="lg" aria-label="Loading" />
```
