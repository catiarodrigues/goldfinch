Toggle individual items on or off in multi-select forms with support for an indeterminate "some selected" state.

## Installation

### Barrel

```tsx
import { Checkbox } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { Checkbox } from "@catiarodrigues/goldfinch/components/checkbox";
```

## Usage

```tsx
import { Checkbox } from "@catiarodrigues/goldfinch";

export default function Example() {
  return <Checkbox defaultChecked aria-label="Accept terms" />;
}
```

## Examples

<LiveCheckbox title="Checkbox" />

Use `checked` and `onCheckedChange` for controlled mode, or `defaultChecked` for uncontrolled mode.

```tsx
<Checkbox defaultChecked aria-label="Accept terms" />
<Checkbox indeterminate aria-label="Some selected" />
```

<Callout type="info">
  The indeterminate state is visual only — the checkbox does not fire a change event when toggled from indeterminate.
</Callout>
