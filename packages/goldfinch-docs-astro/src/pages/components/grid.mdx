Distribute content across preset responsive column layouts — from side-by-side to 6-column grids.

## Installation

### Barrel

```tsx
import { Grid, GridItem } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { Grid, GridItem } from "@catiarodrigues/goldfinch/components/grid";
```

## Usage

```tsx
import { Grid, GridItem } from "@catiarodrigues/goldfinch";

export default function Example() {
  return (
    <Grid variant="3up" gap="base">
      <GridItem className="p-4">1</GridItem>
      <GridItem className="p-4">2</GridItem>
      <GridItem className="p-4">3</GridItem>
    </Grid>
  );
}
```

## Examples

<DemoFrame title="Grid">
  <GridDemo />
</DemoFrame>

Choose a `variant` for the column layout and `gap` for spacing between items.

```tsx
<Grid variant="2up" gap="base">
  <GridItem>Left</GridItem>
  <GridItem>Right</GridItem>
</Grid>
```

<Callout type="info">
  The grid is responsive — columns adjust at md, lg, and xl breakpoints depending on the variant.
</Callout>
