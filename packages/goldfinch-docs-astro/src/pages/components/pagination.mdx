Navigate multi-page content with numbered page buttons, previous/next arrows, and automatic ellipsis for large page counts.

## Installation

### Barrel

```tsx
import { Pagination } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { Pagination } from "@catiarodrigues/goldfinch/components/pagination";
```

## Usage

```tsx
import { Pagination } from "@catiarodrigues/goldfinch";

export default function Example() {
  const [page, setPage] = useState(1);
  return <Pagination current={page} total={20} onChange={setPage} />;
}
```

## Examples

<DemoFrame title="Pagination">

Pass the `current` page, `total` pages, and an `onChange` callback. The component renders page buttons with previous and next arrows.

```tsx
<Pagination current={1} total={20} onChange={(page) => console.log(page)} />
```

<Callout type="info">
  Pagination collapses page numbers with ellipsis when there are many pages to keep the UI compact.
</Callout>
