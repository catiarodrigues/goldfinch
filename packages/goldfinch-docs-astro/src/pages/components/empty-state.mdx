Turn empty lists and search results into opportunities — explain why data is missing and offer a clear next action.

## Installation

### Barrel

```tsx
import { EmptyState } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { EmptyState } from "@catiarodrigues/goldfinch/components/empty-state";
```

## Usage

```tsx
import { EmptyState } from "@catiarodrigues/goldfinch";

export default function Example() {
  return (
    <EmptyState
      title="No results"
      description="Try adjusting your search or filters."
    />
  );
}
```

Provide a `title` and `description` to explain the empty state. Use the `action` slot to render a call-to-action button.

```tsx
<EmptyState
  title="No results"
  description="Try adjusting your search or filters."
/>
```

<Callout type="info">
  Empty states are an opportunity to guide users — always include a helpful message and a clear next action.
</Callout>
