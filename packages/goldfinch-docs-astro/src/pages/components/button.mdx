Trigger actions with five visual variants, four sizes, loading state, and optional icon support.

## Installation

### Barrel

```tsx
import { Button } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { Button } from "@catiarodrigues/goldfinch/components/button";
```

## Usage

```tsx
import { Button } from "@catiarodrigues/goldfinch";

export default function Example() {
  return <Button variant="primary">Submit</Button>;
}
```

## Examples

### Variants

<DemoFrame title="Variants">
  <ButtonDemo />
</DemoFrame>

| Variant | Purpose |
|---------|---------|
| `secondary` | Default button style |
| `primary` | Main call-to-action |
| `destructive` | Destructive action |
| `ghost` | Subtle, no border |
| `outline` | Bordered, transparent |

```tsx
<Button variant="secondary">Default</Button>
<Button variant="primary">Primary</Button>
<Button variant="destructive">Destructive</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="outline">Outline</Button>
```

### Sizes

<DemoFrame title="Sizes">
  <ButtonSizesDemo />
</DemoFrame>

Four sizes available: `xs`, `sm`, `base` (default), `lg`.

```tsx
<Button size="xs">XS</Button>
<Button size="sm">SM</Button>
<Button size="base">Base</Button>
<Button size="lg">LG</Button>
```

### Shape

<DemoFrame title="Shape">
  <ButtonShapeDemo />
</DemoFrame>

Use `shape="square"` for square buttons.

```tsx
<Button shape="square">Square</Button>
```

### Loading

<DemoFrame title="Loading">
  <ButtonLoadingDemo />
</DemoFrame>

Set `loading` to show a spinner and disable the button.

```tsx
<Button loading>Loading</Button>
```

<Callout type="info">
  When loading, the button text remains visible alongside the spinner for context.
</Callout>
