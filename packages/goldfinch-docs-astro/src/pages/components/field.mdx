Wrap any input-like component with a label, optional description, and validation error display for consistent form layout.

## Installation

### Barrel

```tsx
import { Field, Input } from "@catiarodrigues/goldfinch";
```

### Granular

```tsx
import { Field } from "@catiarodrigues/goldfinch/components/field";
```

## Usage

```tsx
import { Field, Input } from "@catiarodrigues/goldfinch";

export default function Example() {
  return (
    <Field label="Email" description="We'll never share your email.">
      <Input type="email" placeholder="you@example.com" aria-label="Email" />
    </Field>
  );
}
```

Pass the input as children and provide `label` and optional `description`. Use `error` to show validation messages.

```tsx
<Field label="Email" description="We'll never share your email.">
  <Input type="email" placeholder="you@example.com" />
</Field>
```

<Callout type="warning">
  When `required` is true, an asterisk is shown next to the label for accessibility.
</Callout>
